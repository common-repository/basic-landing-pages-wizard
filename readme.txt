== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Basic Landing Pages Wizard"
3. **Activate** Basic Landing Pages Wizard from your Plugins page
4. **Click** on the new menu item "Landing Pages Wizard" and step by step wizard!
5. **Click** on Basic Landing Pages menu and see all the generated landing pages
